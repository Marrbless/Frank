# Autonomous Implementation Receipt Template

Use this template after completing one autonomous matrix row. Keep receipts short enough to review quickly and specific enough to resume without guessing.

```md
# Autonomous Implementation Receipt

Date:
Matrix row:
Status: done | blocked | partial

## Scope

- Files changed:
- Behavior changed:
- Behavior intentionally preserved:

## Validation

- Command:
- Result:
- Notes:

## Risks

- Remaining risk:
- Follow-up row:
- Human input needed:
```

Rules:

- Record exact commands, not paraphrases.
- If validation is blocked, mark the row blocked or partial.
- Do not use receipts to hide broad refactors under a small row ID.
